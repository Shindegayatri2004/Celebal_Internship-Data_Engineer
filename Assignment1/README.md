# Shopping Dataset Analysis

## Objective
Perform data cleaning, exploratory data analysis, feature engineering, and visualization using Pandas and Python.

## Dataset
Combined_dataset.csv (1000 rows × 24 columns)

## Operations Performed
- Loaded dataset into Pandas DataFrame
- Explored structure and statistics
- Handled missing values
- Removed duplicate records
- Converted columns into numeric format
- Created derived columns:
  - quantity
  - total_amount
  - price_difference
  - popularity_metric
  - discount_percent
- Performed filtering and category-level analysis
- Created visualizations
- Saved graphs as PNG images
- Exported cleaned dataset

## Key Insights
- Identified top categories.
- Found highest-rated products.
- Determined popular products using rating and ratings count.
- Analyzed price distribution and correlations.

## Output Files
- analysis.ipynb
- cleaned_dataset.csv
- visualization images (.png)
